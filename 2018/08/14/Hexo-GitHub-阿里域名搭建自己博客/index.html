Hexo+GitHub

  • 现在网络上很多，有好多的文章写得很好，在这里推荐两篇文章！只要按照这步骤来，基本上都可以完成!没有什么技术的难度，就是体力活☺☺☺

域名

  • 域名我是在阿里云上买的 阿里云
  • 输入你想要买的域名地址，如图所示!选择一个您觉得合适的域名，加入到清单，然后购买就可以了

    • 购买成功了，进入域名控制台
    • 点击解析设置，然后添加记录，就会出现下面的窗口
    • 设置值，结果如下
  • 这个记录值是什么？如图所示，其实就是 ：https://shimingli.github.io/ 的ip值，当然你需要设置成你的
    • 设置完成了，还有关键的一步，就是实名认证，认证通过了，域名才可以使用。

GitHub Pages的设置

  • 进入到 name.github.io的工程目录下，然后进入到Settings,最后定位到Github Pages的选项,输入你自己购买的域名，我这域名 7块一年，物超所值！

  • 然后把clone的yelee放到themes目录下

  • 在博客的根目录下设置theme: Yelee

遇到的问题

  • Hexo中的Yelee主题，首页不显示文章,需要把themes中的文件_config.yml的search中的on: true的注释去掉，同时必须注释掉onload，同时需要去安装插件!,然后就可以检索了。
